Software

Infotel develops and markets a comprehensive range of software aimed at large organizations, particularly in database management and archiving. The group aims to double its software revenue by 2026.

Arcsys

The Reference Archiving Solution

Arcsys is a software platform specialized in legally compliant electronic archiving and long-term preservation. It enables organizations to store, organize and preserve their digital documents while complying with legal standards (such as NF Z42-013 or ISO 14641).

read more

Main features:

  • Legally compliant archiving (ensuring data integrity and traceability).
  • Document lifecycle management (creation, retention, disposal).
  • Fast search and secure access to archives.
  • Compliance with GDPR requirements and French digital preservation law.
DB/IQ Suite

Excellence in DB2 Management

DB/IQ Suite is a software solution developed to provide monitoring, optimization and intelligent maintenance for databases, primarily IBM DB2 on z/OS.

read more

It is aimed at large enterprises and critical organizations that handle significant data volumes and require continuous availability and optimal performance of their systems.


Main features:

  • Analysis and monitoring of DB2 database performance.
  • Anomaly detection and automated optimization suggestions.
  • SQL query auditing to improve execution speed and reduce costs.
  • Dashboards and detailed reports for system administrators.
iDBA-Online

AI for DB2 Management

iDBA-Online is a tool developed by Infotel for real-time monitoring and automatic management of DB2 databases (on IBM z/OS mainframe environments).

This solution combines continuous analysis, intelligent diagnostics and automated corrective actions, designed for database administrators (DBAs).

read more

Main features:

  • Real-time monitoring of DB2 performance (locks, response times, resource usage…).
  • Automatic incident detection and precise alert generation.
  • Recommendations for corrective actions or automatic interventions to prevent outages.
  • Detailed reports on system activity and usage trends.
  • User-friendly interface to visualize overall database status and simplify diagnosis.
Orlando

The Revolution in Aeronautical Documentation

Orlando (Online Repository for LArge and Numerous DOcuments) is a document management solution specifically designed for the aviation sector.

It enables organizations to digitize their business processes and centralize information in a single system.

read more

Main features:

  • Automatic scanning and recognition of documents (OCR/ICR).
  • Automated classification and intelligent indexing.
  • Management of validation workflows and electronic signatures.
  • Possible integration with other business tools (CRM, ERP, etc.).
Deepeo

Automated GDPR Compliance

Deepeo is a software solution developed by Infotel to help companies manage and protect personal data in accordance with GDPR.

read more

Main features:

  • Automatic mapping of personal data stored in company systems.
  • Detection of duplicates and obsolete data.
  • Implementation of anonymization policies and automatic deletion.
  • GDPR dashboards to monitor compliance in real time.
